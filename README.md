# 🎓 Bright Mind School

A modern PHP + MySQL student registration and admin management system.

## Features

- ✅ Student registration form with validation
- ✅ Admin dashboard protected by login
- ✅ Search, pagination, and delete students
- ✅ Stats: total students, today's registrations, unique courses
- ✅ Modern dark UI with glassmorphism design
- ✅ Session-based authentication with timeout
- ✅ Fully responsive

## Setup

1. Clone the repo
2. Copy `config.example.php` to `config.php`
3. Fill in your database credentials in `config.php`
4. Upload to any PHP host (InfinityFree, Laragon, XAMPP, etc.)
5. Visit `index.php` — the database table is created automatically

## Tech Stack

- PHP 8+
- MySQL / MySQLi
- HTML5 + CSS3 (no frameworks)
- Google Fonts (Inter)
